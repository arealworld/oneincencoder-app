This is a practical open source tool to convert any string to its base64 format. The client (this page) connects to a rest service and receives one by one each value from service response, then each value is writing into the result text field. Try it and have a little fun!, even you're welcome to take a look into the code and help to improve the idea.
The api is currently hosted into a free azure account, so the service seems to hibernate some times to free up resources, so the first call may take a little longer while the service is re-established.
